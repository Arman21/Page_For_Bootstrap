Bootstrap is a free and open-source front-end web framework. It contains HTML and CSS-based design templates for typography, forms, buttons, navigation and other interface components.

As well as optional JavaScript extensions. Unlike many earlier web frameworks, it concerns itself with front-end development only.Bootstrap is the second-most-starred project on GitHub, with more than 131,000 stars.

Bootstrap 4

Mark Otto announced Bootstrap 4 on October 29, 2014.The first alpha version of Bootstrap 4 was released on August 19, 2015.The first beta version was released on 10 August 2017.Mark suspended work on Bootstrap 3 on September 6, 2016, to free up time to work on Bootstrap 4. Bootstrap 4 was finalized on January 18, 2018.
